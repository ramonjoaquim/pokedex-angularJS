# Pokédex

### Tecnologias da stack

- Gulp
- AngularJS 1.8
- BootStrap

### Configurando o ambiente

```bash
cd pokedex-AngularJS && cd src
```

1º
```bash
npm install
```
Task do gulp configurada no projeto

2º
```bash
gulp -T
```

3º
rodar projeto (task gulp default) com reload on change
```bash
gulp
```
